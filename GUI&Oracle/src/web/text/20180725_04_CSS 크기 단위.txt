 + 크기 단위

  - 고정 크기 단위
	 주변요소(부모요소)의 크기에 영향을 받지않고
	일정한 크기를 유지하는 단위

	px : 픽셀, 화소 단위, 정확한 위치나 크기를 설정할 때 사용

	in : 인치, 설정이 되어있는 수치를 따른다(현실세계랑 다름)
	pc : 파이카, 1/6in
	pt : 포인트, 1/12pc, 1/72in

	cm : 센티미터, 1in = 2.54cm
	mm : 밀리미터, 1cm = 10mm



  - 가변 크기 단위
	상대적인 크기를 지정하는 단위

	em : 부모요소에서 지정한 크기만큼 (1em = 부모요소의 크기)
	 ( 주로 부모요소의 line-height 값을 지칭함 )

	ex : 지정된 폰트에서 'x'의 높이 크기만큼

	% : 요소가 들어갈 수 있는 공간에서의 비율

	rem : 최상위(root) 부모요소(<html>)의 크기만큼
