 + 주요 DOM 객체타입

  - Node 객체
	노드를 다루는 기능 + 프로퍼티를 담은 객체

	노드를 탐색, 조작, 타입 파악 기능을 사용할 수 있다
	노드의 계층구조 파악

	DOM의 최상위 객체	


  - Element 객체
	텍스트노드, 주석노드를 제외한 나머지 노드

	태그 노드만 남음

	태그 노드에 대한 기능 + 프로퍼티 제공


  - HTMLElement 객체
	HTML문서에서 사용하는 Element 객체


  - Document 객체
	문서 자체를 나타냄

	Node객체의 하위 객체

	HTML 문서의 root 객체 : <html>
	XML 문서의 root 객체


  - HTMLDocument 객체
	HTML 문서용 Document 객체

	<html> 태그 요소

	HTML 문서의 모든 요소(태그)를 조작할 수 있게 하는 객체
