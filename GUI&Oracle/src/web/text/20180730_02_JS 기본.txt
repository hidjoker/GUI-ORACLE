
 + 자바스크립트, Javascript
  - HTML문서의 동작의 표현하는 언어

	ex)
	  <form>에 입력한 값이 적절한지 판단 (validation)

	  <button>을 클릭했을 때 이벤트 처리 (event listener)

	  <form>의 submit을 수행 (form data submit)


  - HTML문서의 <head> 내에 <script>태그 안에 작성함

  - 브라우저의 자바스크립트 엔진에 의해 인터프리트방식으로 동작

  - 자바스크립트 파일(.js)을 HTML에서 로드해서 적용하기도 함

  - 자바스크립트를 허용하지 않는 브라우저에서는 작동 안 함

  - HTML파일안에 포함되어 클라이언트(브라우저)에 전달됨
   ( 패스워드가 적혀있거나 민감정보가 코드에 포함되는 경우는
    없어야한다 )
