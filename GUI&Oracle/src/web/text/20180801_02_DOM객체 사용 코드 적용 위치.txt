 + DOM객체를 사용하는 코드의 적용 위치

  - DOM객체는 <body>의 계층구조를 모두 확인한 이후에 사용
   해야하므로 DOM객체 구성이 완료되는 시점까지 DOM객체 사용을
   늦춰야한다
	-> 브라우저가 <body>의 내용을 절차적으로 다 읽은 이후

	-> 브라우저가 html문서를 다 읽은 이후 onload 이벤트 발생


  1. <head> 태그 내 <script> 태그에서 window.onload 리스너로 작성

  2. </body> 닫는 태그 바로 위에 <script>로 바로 작성
	(window.onload 필요 없음)

  3. 일반 함수에서 사용하고 태그의 이벤트 리스너로 호출하기
