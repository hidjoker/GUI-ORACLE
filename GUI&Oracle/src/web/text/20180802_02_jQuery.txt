 + jQuery
  - 자바스크립트 DOM을 사용하기 쉽게 문법을 정의한 라이브러리

  - 순수 자바스크립트 코드보다 효율적, 코드생산성이 향상됨

  - HTML노드, 스타일, 속성, 이벤트 등을 다룰 수 있다

  - CSS 선택자를 이용해서 노드를 쉽게 찾을 수 있다

  - http://www.jquery.com/



 + jQuery 주요 기능
  - jQuery DOM : JS DOM객체를 다루는 기능

  - jQuery Ajax : Ajax(비동기 통신) 기능

  - jQuery Event : JS Event 다루는 기능


  - jQuery Plugin : 추가 기능(플러그인)

  - jQuery Effects : 효과(애니메이션, 모션 등)



 + jQuery 1.x, 2.x, 3.x

  - 1.x 다양한 브라우저에 호환되어 있음
  - 최대한 많은 브라우저에 호환되는 게 목적
   (브라우저 특징에 맞는 동작이 생략되어있음)

  - 2.x 코드 성능, 다양한 기능을 제공하기 위해서 IE8이하는 지원 X
  - 옛 코드 버리고 가볍고 빠르게 만들고 기능도 추가함

  - 3.x 는 2.x의 업그레이드 버전
  - jQuery Compat 3.x 버전은 1.x의 업그레이드 버전
