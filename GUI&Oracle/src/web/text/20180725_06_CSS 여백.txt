
 + CSS 여백
  - 컨텐츠(내용물)을 감싸고 있는 투명한 공간

  - top right bottom left의 영역으로 구성되어있음

  - 여백 공간의 크기는 주로 auto 또는 %단위 또는 길이단위로 설정


  - margin, 외부여백
	테두리(border)의 바깥쪽 영역

	margin : all;

	margin : top&bottom left&right;

	margin : top left&right bottom;

	margin : top right bottom left;



  - padding, 내부여백
	테두리(border)와 컨텐츠(element) 사이의 공간

	padding : all;

	padding : top&bottom left&right;

	padding : top left&right bottom;

	padding : top right bottom left;
