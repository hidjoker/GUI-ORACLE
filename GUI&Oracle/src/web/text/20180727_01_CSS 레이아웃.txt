 + CSS 레이아웃
  - 요소의 크기, 위치, 배치 등을 관리하는 스타일 속성


 + 레아아웃 크기
  - width : 요소의 너비
  - height : 요소의 높이
   ( px, %, auto 가 많이 사용됨 )

  - max-height, max-width, min-height, min-width
	요소의 최대or최소 너비or높이를 설정하는 속성


 + 레이아웃 여백
  - margin : 외부 여백
  - padding : 내부 여백


 + 레이아웃 테두리
  - border : 요소의 테두리
