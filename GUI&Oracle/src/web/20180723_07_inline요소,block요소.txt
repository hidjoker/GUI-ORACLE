
 + inline 요소, block 요소
  - 태그의 영역을 지정하는 방식

  - 태그마다 기본적으로 inline 또는 block 속성을 주로 받는다


  - 인라인요소("display: inline")

	컨텐츠(내용물)의 크기만큼만 공간을 차지한다

	동일한 줄에 옆으로 배치되며 표현된다

	ex) 단순 텍스트, <span>, <a>, ...


  - 블락요소("display: block")

	한 줄 전부를 공간 차지한다

	인라인요소가 들어갈 공간을 만들어주는 요소

	ex) <div>, <p>, <h1>, ...
